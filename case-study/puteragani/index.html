01 / Overview

The assignment was technical completeness, not a visual redesign.

PuteraGani.com is an owned editorial website with static article pages, category pages, and a homepage whose article discovery depends partly on JavaScript and structured data.

The implementation focused on the invisible systems used by crawlers and sharing platforms: canonical targets, initial HTML, page metadata, JSON-LD, sitemaps, robots controls, redirect rules, image dimensions, and compiled production CSS.

Portfolio disclosure

This is my owned website, not a paid client engagement. It demonstrates implementation capability and audit outcomes. This case study does not claim traffic or ranking growth without verified Google Search Console data.

02 / Implementation

Six systems changed together.

  1. 01

    Canonical routing aligned with live clean URLs

    Canonical targets were standardized around the final public paths rather than raw .html source filenames, reducing duplicate-path ambiguity.

    Indexation
  2. 02

    Metadata and social previews made consistent

    Titles, descriptions, Open Graph, Twitter cards, and page-specific copy were brought into a repeatable template-driven system.

    On-Page
  3. 03

    Structured data matched each page purpose

    WebSite, Person, Organization, BlogPosting, BreadcrumbList, CollectionPage, AboutPage, and ContactPage markup were used where appropriate.

    Schema
  4. 04

    Critical discovery moved into initial HTML

    Key article links and featured content were made available before JavaScript execution, improving crawler and no-script discovery.

    Rendering
  5. 05

    Crawl controls and production assets were added

    The release added an XML sitemap, robots rules, redirect and header controls, llms.txt, image dimensions, and compiled production CSS.

    Technical
  6. 06

    A repeatable validation loop closed the work

    The final state was checked across every production HTML page instead of relying on a single homepage score.

04 / Honest limitations

What these scores do not prove.

01

No ranking-growth claim yet

Technical completion is a foundation. Query, impression, click, and position growth require Search Console data over time.

02

Backlink authority remains weak

SEOptimer graded Links at F, while another audit found only one referring domain. Off-page authority is the clearest next priority.

03

Audit tools disagree

Rank Math reported 95/100, SEO Site Checkup 85/100, and SEOptimer an overall B. Each tool uses a different model.

04

Some recommendations are contextual

Local Business schema and a Google Business Profile are not automatically appropriate for an editorial website.
